# Standard Operating Procedure (SOP)

1. Fabrication
- Tear-and-stack tBLG
- hBN encapsulation
- Baseline Raman

2. Ni Supply
- Pre-deposition thickness sweep
- UHV anneal (450–550°C, ≤1e-9 mbar)

3. Screening
- Raman (G/2D shift, I(D)/I(G))
- Optional Hall

4. Measurements
- 4-probe R(T)
- I-V
- R(H,T)
- SQUID / AC susceptibility

5. Controls
- Ni=0 control
- Off-angle control
- Electrode swap
- Geometry swap
- Thermal cycling
